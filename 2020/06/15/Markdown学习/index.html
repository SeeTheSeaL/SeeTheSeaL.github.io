markdown学习

标题

#+空格+标题名表示一级标题

##+空格+标题名表示二级标题

###+空格+标题名表示三级标题

。。。。

最多支持六级标题

字体

Helloworld！

**加粗

Helloworld！

*斜体

Helloworld！

***斜体+加粗

Helloworld！

~~废弃

引用

内容。。。。。。。。

大于号+空格(一般引用别人的文章时用)

分割线

—/***三个减号或三个星号

图片

![名字]+(可以放本地或者网络图片)

加号不需要，这里有加号是为了格式不乱

超链接

超链接名字

[超链接名字]+()

加号不需要，这里有加号是为了格式不乱

列表

  1. A
  2. B
  3. C
    • A
    • B
    • C

1.空格(表示有序列表)

-空格(表示无序列表)

表格

名字 性别 生日
张三 男 1997.1.1

名字|性别|生日

–|–|–|

张三|男|1997.1.1

(需要打开源代码模式，删除每行之间的空格)

更快的方法:直接在段落–>表格–>插入表格(快捷键option+command+T)
